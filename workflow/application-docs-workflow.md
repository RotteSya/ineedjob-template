# Application Documents Workflow（応募書類作成ワークフロー）

Last updated: 2026-06-08

## Purpose

このワークフローは、**履歴書・職務経歴書・送付状・封筒**を毎回同じ品質で作るための標準手順です。厚生労働省／ハローワーク発行の3冊（令和7年5月発行）――「応募書類の作り方」「職務経歴書の作り方」、および別冊ワークブック「職務経歴書作成のためのマスターシート」――の方法論を抽出し、本リポジトリの正本ファイルに接続して運用できる形にまとめたものです。

`workflow/company-interview-workflow.md` が「面接稿を作る」標準手順であるのに対し、本書は「提出書類を作る」標準手順です。応募の流れとしては **本書（書類選考突破）→ workflow/company-interview-workflow.md（面接突破）** の順で使う。

新しい対話で「書類を作りたい」「履歴書／職務経歴書を直したい」という依頼が出たら、最初に本書と以下の正本を読む:

- `profile/candidate-profile.md` … 候補者の定位・事実・包装上の注意点（正本）
- `profile/職務経歴書.md` … 全経験を網羅した母版（フリースタイル／キャリア式構成）
- `profile/PR.md` … 自己PRの母庫
- `profile/career-criteria.md` … 就活軸の母軸
- `profile/evidence-ledger.md` … 数字・受賞・実績の根拠台帳

## 0. 全体像 — 4書類の役割と優先順位

| 書類 | 位置づけ | 役割 | 応募先ごとに変えるか |
| --- | --- | --- | --- |
| 履歴書 | **審査書類**（応募書類かつ採否判断に使う） | 学歴・職歴・資格を「ルール通り」に正確に。最初に見られ、ここで落ちると職務経歴書は読まれない | 志望動機・アピール欄は変える |
| 職務経歴書 | **実務能力のアピール書類** | 「自分には〜の実務能力・長所・強みがあり貢献できる」を自由様式で詳しく示す | **毎回変える**（使い回し不可） |
| 送付状（添え状／カバーレター） | ビジネス添え状 | 書類束の一番上に付ける挨拶状。メール送付時は本文が代わり | 宛先・応募経緯を変える |
| 封筒 | 容れ物 | 折らずに入れる（角2型）。第一印象に直結 | — |

原則:

- 採用選考は「面接の前にまず書類選考」が多い。**面接官に「会ってみたい」と思わせる**ことが書類の最優先目的。
- 読み手（採用担当者）の立場に立ち、誤字脱字ゼロ・ルール厳守・読みやすく丁寧に。書類の雑さは「仕事が雑」「意欲がない」と読まれる。
- 完成書類は必ず**コピーを保管**し、面接前に見返す（面接は書類の記載内容に基づいて質問される）。

## Phase A. マスターシート（素材集）を整える — 一度作って使い回す

ワークブックの中核思想は「応募先ごとに職務経歴書をゼロから作るのは大変なので、**自分の職務経歴・能力を全て網羅した素材集（マスターシート）**を先に作り、そこから応募先ごとに取捨選択する」こと。

**本リポジトリでは、マスターシートはすでに以下のファイル群が担っている。** ワークブックのシート①〜⑥を本リポジトリのファイルへ対応づけて運用する:

| ワークブックのシート | 内容 | 本リポジトリでの対応先 |
| --- | --- | --- |
| シート① 勤務先情報 | 期間・企業名・事業内容・従業員数・所在地・退職理由（6社分） | `候補者プロフィール`の経歴 / `profile/職務経歴書.md` の「職務経歴」 |
| シート② 職務内容・実績 | 時期／入社・配属・異動・昇格・退社／勤務先／会社概要／所属（人数）／役職（部下数）／職務内容／実績、および特別な経験 | `profile/職務経歴書.md` の「職務経歴」「主要プロジェクト」、数字は `profile/evidence-ledger.md` |
| シート③ 職務以外の活動 | 社外活動（勉強会・異業種交流会）、社会活動（PTA・町内会・サークル・ボランティア） | `profile/職務経歴書.md` の「学生時代の活動」、`profile/candidate-profile.md` |
| シート④ 能力を示す事実 | 取得資格／PCスキル／語学／活かせる技術・技能・知識／人脈、＋業務関連の学歴・職業訓練・社内研修・自己啓発 | `profile/職務経歴書.md` の「資格・語学」「使用可能ツール・スキル」 |
| シート⑤ 能力・長所・強みのキーワード化 | ①〜④から一言キーワードへ | `profile/PR.md`、`profile/career-criteria.md` |
| シート⑥ 応募先との適合性・自己PR設計 | 応募先ごとの要件収集と自己PR | 下記 Phase B（workflow/company-interview-workflow.md の Step 1.5/4/7 と共有） |

### A1. シート②で職務経歴を振り返るときの8観点

各職務について、以下を具体的に書き出す（アルバイトも対象になりうる）:

1. 時期
2. 入社・配属・異動・昇格・退社の別
3. 勤務先
4. 会社概要（事業内容・従業員数）
5. 所属（所属部署の人数）
6. 役職（部下の人数）
7. 職務内容（どんな仕事をしたか）
8. 実績（**どんな経過で／どんな成果を**。数値化できるものは数値で。難しければ創意・工夫・改善点）

さらに「通常は経験できない特別な経験」――受賞歴、プロジェクト経験（新規開店・大事業・業務改善・機構改革）、イベント経験（大きな催し・株主総会・決算棚卸し）、特命業務――があれば必ず書き出す。

### A2. アピールが見つからない時の4つの発掘法

職歴からアピールが出にくい場合（職歴が短い・ブランクがある・新卒枠の場合は特に重要）、次の4視点で書き出すと潜在的な強みが見つかる:

1. **仕事の中で得たもの** … 「心がけたこと」「得たもの」「身につけたもの」「できるようになったこと」
2. **努力・頑張り・乗り越えたこと** … どうやって努力し・頑張り・乗り越えたか
3. **前向きなエピソード** … やりがい・喜び・できたこと・ほめられたこと・達成感・成長
4. **各職務の共通項** … 共通して「できること」「得意なこと」「自信のあること」「やりたいこと」

→ 出てきた要素は `profile/PR.md` / `profile/career-criteria.md` / `profile/evidence-ledger.md` に還元し、母版を厚くする。

## Phase B. 応募先ごとの設計（ワークブック シート⑥）

職務経歴書は応募先ごとに変えるのが原則。**応募先が求めることを把握 → 自分の素材から最大限アピールできる内容を選ぶ。**

1. **求人票・採用ページから要件を収集**し、項目別に分解する: 職種／仕事の内容／学歴／必要な経験／必要な免許・資格／その他。
2. **応募条件クリア判定**: 自分が各条件を満たすか確認。満たさない項目は、シート②〜④（＝`profile/職務経歴書.md`／`profile/evidence-ledger.md`）から「求めることの一部や類似する経験」を探し、アピール要素に転用する。
3. **Hidden Need の推測**: 応募条件に書かれない期待を、業界・職業の動向（ネット・書籍）から推測し、シート⑤の能力・長所・強みで応えられないか考える。
4. **応募理由の整理**: なぜこの応募先を選んだか／応募先内で自分の経験・能力を活かせる分野はどこか。これが志望動機の土台になる。

> この Phase B は `workflow/company-interview-workflow.md` の **Step 1.5（就活軸・4層志望動機・企業決定軸）／Step 4（要件抽出 Must/Want/Hidden/Risk/Hook）／Step 7（Fit Map）** と同一の素材を使う。同じ会社で書類と面接を準備する場合は、`docs/<company>/` の基本ファイルへ一度だけ要件抽出を行い、書類・面接の双方から参照する（二重作業をしない）。

## Phase C. 職務経歴書の作成

### C1. 作成の基本ルール

**様式・用具**

- A4縦サイズ・白無地の紙 1〜2枚程度。パソコンで横書きが一般的（別紙の工事経歴一覧・作品集等を付ける場合は枚数増加可）。
- 手書きも可。黒のボールペン・万年筆（鉛筆・シャープペンシル不可）。手書きは罫線入り用紙が書きやすい。

**必須項目と項目数**

- **必須**: 冒頭の「標題」「氏名」「日付」＋「職務経歴」。
- そのほかは自由だが、自分をアピールできる項目（取得資格・パソコンスキル・活かせる能力・自己PR・志望動機 等）を選んで追加。
- 記載項目数は **冒頭部分と「職務経歴」の下位項目を除いて全部で3〜6項目程度**が一般的。

**読みやすい文章表現**

- 一文は長くしすぎない。**原則は体言（名詞）止め**（「〜を担当」「〜に従事」）。ただし「自己PR」「志望動機」は「〜しました。」など丁寧な文体にする。
- 内容が多すぎると焦点がぼける。メリハリをつけ、アピールしたい点が的確に伝わるようにする。
- 年号は和暦か西暦に統一（通常は和暦／**外資系・IT企業は西暦**が一般的）。
- 企業名・資格名・学校名などの固有名詞は略さず正式名称（×「（株）○○」→ ○「株式会社○○」）。読み手が分かりにくい専門用語は（　）書きで説明を付す。
- 作成後は必ず読み返し、誤字脱字、パソコンなら印字のかすれ・汚れを点検。

**見やすいレイアウト**

- 本文の文字サイズは標題・見出しを除き **10.5〜12ポイント**程度。本文フォントは統一。
- 行間・字間を詰めすぎない。上下左右に十分な余白（**特に左余白は採用担当者のファイリング用**に必要）。
- 標題・見出しを強調してメリハリ: 大文字・太字・下線、冒頭に「1.」「2.」等の番号、■●等の装飾、＜　＞や【　】で囲む。一部は①②や※・で箇条書き、表形式も可。

### C2. 記載項目の選定（A冒頭 ＋ B〜E から3〜6項目）

冒頭はAを記載。さらにB〜Eから3〜6項目程度を選び、見出しを付けて中身を書く。**太字＝必須またはよく用いる項目。**

| 区分 | 項目（見出し例） | 記載上の留意事項 |
| --- | --- | --- |
| **A 冒頭** | **標題** | 必須。「職務経歴書」を大きめの字で最上部中央に。 |
| | **日付** | 必須。提出日（郵送=投函日／持参=持参日）。履歴書と同じ日付。和暦か西暦で統一。 |
| | **氏名** | 必須。パソコン作成でも氏名だけ自筆でも可。 |
| | 年齢／捺印／住所／電話番号 | 任意。捺印は氏名の右横。電話は連絡の便のため記載推奨。 |
| **B 応募職種** | 応募職種（志望職種／希望職種） | 複数職種募集時など、求人票の職種名をそのまま簡潔に。 |
| | 職歴概要（略歴／経歴概略／要約） | 職務経歴のエッセンスを先頭で要約。 |
| **C 経験関係** | **職務経歴** | 必須。各職務ごとに以下を記載。 |
| | 時期／勤務先／事業内容／所属／役職／雇用形態／**職務内容**／実績 | 勤務先は正式名称。職務内容＝仕事の具体的内容・処理方法・責任範囲・顧客/製品/数量/使用機器/技術。実績は可能なら数値。雇用形態（派遣・契約・パート等）は（　）書き。 |
| | 退職理由 | 「一身上の都合により退職」等簡潔に。詳しく書く場合は前向きに、前職批判は厳禁。 |
| **特別な経験** | 受賞／活かせる経験（プロジェクト・イベント・特命業務） | 表彰・コンクール受賞、新規開店・業務改善、株主総会・決算棚卸し、上司代行 等。 |
| **職務以外の活動** | 社外活動／社会活動／学生時代の活動／趣味・スポーツ | 勉強会・異業種交流、PTA・ボランティア、ゼミ・サークル。職務能力や適性に結びつくもの。 |
| **D 能力関係** | 取得資格／パソコンスキル／語学力 | 正式名称・取得年月・水準。失効・勉強中・1次合格・2次待機でも明記の上で可。 |
| | **活かせる能力** | よく用いる。技術・技能・知識でアピールできること。見出しを能力名にしてもよい。 |
| | 人脈／業務関連の学歴／職業訓練／社内研修／自己啓発 | 職務に直結する学習歴・研修歴。 |
| **E その他** | **自己PR** | よく用いる。長所・強み（性格・行動特性、仕事への姿勢・意欲、将来目標）。 |
| | **志望動機** | 履歴書と同内容にせず、エピソードを交えて補足（「職務経歴」より前に置くことが多い）。 |

### C3. 「職務経歴」の記載スタイルを選ぶ

| スタイル | 内容 | 向くケース |
| --- | --- | --- |
| **編年体式**（デフォルト） | 古い経歴から時系列で並べる。迷ったらこれ。 | 一般的。経歴の流れを見せたい |
| **逆編年体式** | 新しい経歴から並べる。 | 直近の経験を強く見せたい／長期勤続→初転職 |
| **キャリア式** | 職務内容の類型ごとに区分し、各区分内をアピール力の高い順に。職歴の時間的流れが分かりにくくなるので、別に時系列の「略歴」を併記する。 | 転職が多い／職歴が多く複数に分類できる |
| **フリースタイル** | 類型ごとに区分した上で各区分を時系列に並べる、勤務先ごとに職務内容・実績を書く 等。 | 派遣・アルバイト経験中心 など |

> **母版 `profile/職務経歴書.md` はフリースタイル／キャリア式構成を基本とする**（応募想定職種→職務要約→キャリアの方向性→活かせる経験・強み→職務経歴→主要プロジェクト→個人プロジェクト→…→自己PR）。応募先に合わせて編年体寄り／プロジェクト強調へ重心を変える。

### C4. 状況別アピール戦略（公式の8記載例）

応募者の状況により「最大のアピールポイント」が変わる。母版から子版を作るとき、まず本人の状況に最も近い型を選ぶ:

| # | 状況 | スタイル | アピールの核 |
| --- | --- | --- | --- |
| ① | 同職種への転職 | 編年体式 | **即戦力（経験者）**。前職の職務内容・実績・知識能力を具体的に |
| ② | 異職種への転職 | 編年体式 | ①なぜ異職種へ ②応募先の仕事を理解した上で応募 ③こなせる能力・適性 の3点を説明（一般に「未経験者」扱い） |
| ③ | 転職が多い | キャリア式 | 前向きな側面（能力の幅・各社での実績・好奇心）＋**勤続意欲** |
| ④ | 派遣就業中心 | フリースタイル | 専門的な知識・能力。社員勤務へ**応用・活用できる要素**を抽出 |
| ⑤ | アルバイト・パート中心 | フリースタイル | 責任ある仕事を任された経験。社員勤務へ応用できる要素 |
| ⑥ | 職歴にアピール要素が少ない | 編年体式 | **適性・潜在能力・将来性**。興味分野・行動特性・性格が職種に向くこと。学生時代/社会活動からも |
| ⑦ | 長期勤続→初めての転職 | 逆編年体式 | 社内だけでなく**他社でも通用・応用できる**能力・知識・経験 |
| ⑧ | 職歴が短い | 編年体式 | ビジネスマナー・基礎的実務能力・高い意欲。**短期退職の理由を採用担当者が納得する形で説明** |

> **適用の目安**: 留学生・新卒で職歴が短い／ブランクがある場合は、**②異職種転職寄り × ⑥潜在能力・将来性アピール**のハイブリッドになりやすい。型を当てる際は必ず `profile/candidate-profile.md` の Packaging Warnings と、ブランク・新卒枠の説明方針を確認する（具体的な禁止項目は profile を正本とする）。

### C5. 「能力・長所・強み」のキーワード化

公式パンフ17〜18頁のキーワード集（2系統）を使い、シート⑤で言語化したものを「活かせる能力」「自己PR」へ落とす:

- **性格・行動特性系**: 積極的／チャレンジ精神／向上心／プラス思考／明るい／社交的／負けず嫌い／粘り強い・忍耐強い／発想力・好奇心／思いやり・協調性・気配り／素直・誠実・几帳面／探求心 など。
- **ビジネス能力系**: 企画力・先見力・情報収集力／理解力・課題認識力・問題解決力・論理的思考力／交渉力・チームワーク・指導力・コミュニケーション能力／説得力・プレゼンテーション能力・文章力／迅速性・正確性・計画性・実行力・報連相 など。

各キーワードは抽象語のままにせず、**「→（具体例の一文）」の形で証拠とセット**にする（例:「正確性 →ミスのない正確な業務遂行が生産性を高めることを学んできました」）。`profile/PR.md` から主PRを1つ選び、`profile/evidence-ledger.md` の数字で裏づける。

## Phase D. 履歴書の作成

### D1. 基本姿勢

- 履歴書は**審査書類**。学歴・職歴・資格の内容に加え、様式選び・筆記用具・文字サイズ・文体など**作成方法のルール**が重要。
- 近年は職務経歴書の提出も求められるが、採用担当者は**まず履歴書を見る**。履歴書だけでもアピールできるようにする。書類選考突破を最優先に内容を練る。

### D2. 様式・用具

- 様式は「JIS規格履歴書」が標準。大卒用・転職者用・パート用等の市販様式も可。自分のアピールしたい内容が書きやすいものを選ぶ。
- 大きさは A4版・B5版。企業書類はA版統一が多く記載欄も広い**A4版を推奨**。応募先指定があれば従う。
- **「保護者」欄のある旧版は使わない**。最新版はハローワークインターネットサービス／厚労省様式例を利用。
- 手書きかパソコンかは応募先による。現在は「どちらでも可」が多い。HPからエントリー可能な企業や**IT関連・外資はパソコン推奨**。
- 手書きは黒のボールペン・万年筆（鉛筆・シャープ不可）。**修正液・修正テープ・二重線・訂正印での修正は厳禁 → 最初から書き直す**。コピーや返却された使い回しは印象を悪くするので不可。

### D3. 読みやすく丁寧に

- 楷書体で丁寧に、枠いっぱいに詰めない。字の大きさは揃える（氏名はやや大きく、ふりがな・電話・郵便番号はやや小さく）。
- 文体は「です・ます」調で統一。年号は和暦か西暦に統一。
- 略号禁止（「〃」「同上」「同左」を使わない）。固有名詞は正式名称。名称変更があれば当時の名称＋「（現・○○）」。
- 作成後は必ず読み返す。身近な人に見てもらうのも有効。

### D4. 写真

- 本人のみ正面上半身・無背景・無帽、自然な表情。スナップ不可。概ね6か月以内。カラーが一般的。
- 裏面に氏名を記載し、のりでしっかり貼付（セロハンテープ不可、市販の両面テープは可）。
- 身だしなみは基本スーツ、襟付き。男性は白／淡色シャツ＋ダーク系スーツ＋地味なネクタイ。女性は透ける生地・過度な装飾・目立つアクセは避ける。面接で眼鏡をかける予定なら眼鏡で撮影。

### D5. 記載項目の要点（①〜⑪）

1. **日付** … 記載日でなく提出日（郵送=投函日／持参=持参日）。
2. **生年月日** … 年号は和暦か西暦に統一。
3. **現住所** … 都道府県・建物名・部屋番号を省略せず正式表記（「1-2-3」→「一丁目2番地3号」）。
4. **電話** … 固定・携帯の両方あれば両方。
5. **連絡先** … 現住所以外で常時連絡が取れる場合のみ。在職者は現勤務先を書かない。e-mail可。
6. **学歴** … 入学・卒業を古い順。高校以上は学部・学科・コース・専攻も。職業訓練も記載可。予備校は通常書かない。
7. **職歴** … 入社・退社を古い順が基本。異動・昇格・出向、雇用形態（正職員以外は（　）書き）、職務経歴書を出さない場合は事業内容・従業員数・所属部署・職務内容を付記。退職理由は簡潔に。前向きな理由は「志望動機・アピール」欄で丁寧に。職歴の最後に「以上」。
8. **免許・資格** … 「○○免許 取得」「○○検定 合格」。**応募先へのアピール力の高い順**を推奨。失効・勉強中・1次合格でも明記の上で可。資格がなくても活用できる技能は「※パソコン（Word, Excel）を実務で操作可能」等と付記可。
9. **志望の動機・アピールポイント** … D7参照（最重要欄）。
10. **特技** … 応募先で活かせる技能を優先。個人的活動も仕事に関係するものを優先。
11. **本人希望記載欄** … 勤務条件はストレートに書かず面接で相談。空欄不可（「貴社の規定に従います」「○○職を希望します」等）。

### D6. 職歴欄の記載パターン

- **一般**: 「○○会社 入社／退社」。丁寧版は（事業内容・従業員数）、配属、職務内容（一日平均○件 等）を付記。
- **職業訓練**: 実施機関の正式名称（現名称を併記）＋科名（公共職業訓練）入所／修了。
- **派遣**: 派遣元・派遣先・業務を明記（複数先・専門性も記載可）。
- **アルバイト/パート**: 学業期間中は通常書かないが、卒業後や応募先の職務に関係する／責任ある仕事／マナー習得を示せる場合は「（アルバイト）」と明記の上で記載。複数まとめ記載も可。
- **在職中**: 最終行に「現在に至る」「現在に至る（令和○年○月末退職予定）」等。
- **会社以外**: 学校・病院・個人事業・団体・官公庁は「勤務（入職）／退職」。個人事業主は「開業／廃業」、会社経営者は「設立／解散」、家業手伝いは「家業である○○業に従事」。

### D7. 「志望の動機・アピールポイント」（最重要欄）

3要素を盛り込む:

1. **応募先を選んだ理由**（なぜその会社か）
2. **自分の経験・能力のアピール**（自分のどこを仕事に活かせるか）
3. **意欲**（どんな仕事でその会社に役立ちたいか）

書き方:

- まず求人票・店舗・製品・広告・HPで応募先をよく知る。その上で応募先ごとに①〜③の具体内容を検討。
- アピールは**性格・行動特性が表れたエピソード**で裏づける。面接で最もよく質問される欄なので、面接でどう説明するかをイメージして書く。
- 「貴社で学びたい」「充実感を得たい」等の一方的な希望だけでは弱い。具体的で説得力のあるメッセージを。
- 公式の骨組み例（「私はこれまで〜を志し…貴社の〜に魅力を感じ…〜の面では誰にも負けない自信があります…」等）を自分の言葉でアレンジ。

**敬称**: 文書では「貴社」（口語の「御社」は使わない）。会社以外は ―― 財団/社団/NPO/社福/独法=貴法人・貴団体、協会=貴協会、組合=貴組合、銀行=貴行、学校=貴校（大学・短大は貴学可）、幼稚園/保育園=貴園、病院/医院=貴院、会計/法律事務所=貴事務所、官公庁=貴省/貴庁/貴局/貴所/貴役所/貴役場、商店/飲食店=貴店、個人事業主=（事業所名）様。迷う場合は「（法人名）様」。

> 履歴書の志望動機（短い・要約）と職務経歴書の志望動機（詳しい・エピソード）は**同一文の写しにしない**。`profile/career-criteria.md` の軸 → `profile/PR.md` の主PR → `workflow/company-interview-workflow.md` Step 1.5 の4層ロジックで一貫させた上で、媒体ごとに長さと粒度を変える。

## Phase E. 送付状（添え状／カバーレター）の作成

- A4縦・白無地1枚にパソコン横書きが一般的（手書きなら縦書きの手紙形式も可）。
- **メールで送る場合はメール本文が送付状の代わり**。
- 履歴書をB5で作り職務経歴書（A4）を添付しない場合は、送付状もB5に揃える。

**構成要素**（頭語で始め結語で終える）:

1. ① 頭語・前文（拝啓　時下ますますご清栄のこととお慶び申し上げます。）
2. ② 応募の経緯（「ハローワーク□□の紹介により貴社の○○職に応募」等）
3. ③ 志望理由・自己PR（任意。中身は履歴書/職務経歴書に書くのが基本なので**省略可**）
4. ④ 提出する応募書類の種類（箇条書き可）
5. ⑤ 面接・採用検討のお願い
6. ⑥ 結語（敬具）

**宛先**は3段が基本: 「応募先企業名（正式名称）」→「採用担当者の所属部署と肩書き」→「採用担当者名」。担当者名が分からなければ、部署名の右に「御中」、または「採用ご担当者様」。

日付は記載日でなく**履歴書・職務経歴書と同じ提出日**。差出人欄に住所・電話・メール・氏名（氏名のみ自筆や押印も可だが必須ではない）。

## Phase F. 封筒・提出

**郵送**

- 書類は上から **送付状 → ハローワーク紹介状 → 履歴書 → 職務経歴書** の順に重ねる。
- 折らずに入れる（**角2型**・白／薄茶）を推奨。ゼムクリップ留めやクリアファイルで崩れ防止。やむを得ず折る場合は長3型にきれいに三つ折り。
- 表面左側に**「応募書類在中」を朱書き**。封はのり付けし**「〆」**（×でない）を記す。セロハンテープ・ホチキス止めは厳禁。
- **郵便料金の不足は致命的に印象を損なう**。不明・不安なら郵便局窓口から投函。紹介を受けたらすぐ送る。

**持参**

- 角2型の折らない方式。郵送に準じて表裏を記載（応募先住所は除く）。切手・封は不要。
- 訪問前に電話で訪問時間・採用担当者を確認。すぐ面接になる場合もあるので、身だしなみを整えて訪問。

## （参考）暦年の換算

**和暦→西暦**: 昭和年＋1925／平成年＋1988／令和年＋2018（例: 令和元年=1+2018=2019）。

**桁あふれの繰り上げ**: 昭和が64年以上 → 昭和年−63=平成年。平成が30年以上 → 平成年−30=令和年（平成31年は4月まで、令和元年は5月から）。

**学校の入学・卒業年**（生年に加算。1/1〜4/1の早生まれは「生年−1」に置換）:

- 小学校入学 ＝ 生年＋7
- 中学校入学 ＝ 生年＋13
- 高等学校入学 ＝ 生年＋16
- 大学・専門学校入学 ＝ 生年＋19
- 大学・専門学校（2年制）卒業 ＝ 生年＋21／（3年制）＋22／（4年制）＋23
- 留年・浪人があればその年数を加算。
