## Company-Specific Selection Rule

企業別に自己PRを書く時の順番:

1. 求人票の求める人物像を読む。
2. `profile/career-criteria.md` の母軸から、その会社に合う軸を2-3個選ぶ。
3. このファイルから主PRを1つ選ぶ。
4. 補助PRを1つだけ選ぶ。
5. 会社固有の言葉に変換する。
6. 事実は `profile/candidate-profile.md` と矛盾させない。

避けること: 一つの面接で強みを出しすぎる / 会社ごとにまったく別の自己PRにする / 志望動機と自己PRが別方向になる / 経験を過大評価させる言い方にする。
